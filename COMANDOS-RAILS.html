Lista de Comandos Rails

O Rails possui uma ampla gama de comandos que facilitam o desenvolvimento de aplicações web. Aqui está uma lista com os principais comandos e suas funções:

Comandos Básicos

Comando Descrição Documentação
rails new [nome_app] Cria uma nova aplicação Rails. Documentação
rails server Inicia o servidor de desenvolvimento. Documentação
rails console Abre um console interativo para testar objetos e métodos. Documentação
rails generate [tipo] [nome] Gera arquivos como modelos, controladores, migrações, etc. Documentação

Comandos de Banco de Dados

Comando Descrição Documentação
rails db:migrate Aplica migrações de banco de dados. Documentação
rails db:create Cria o banco de dados especificado na configuração. Documentação
rails db:drop Remove o banco de dados. Documentação
rails db:seed Popula o banco de dados com dados iniciais. Documentação

Comandos Avançados

Comando Descrição Documentação
rails dbconsole Abre o console do banco de dados. Documentação
rails destroy [tipo] [nome] Remove arquivos gerados anteriormente. Documentação

Comandos de Configuração

Comando Descrição Documentação
rails credentials:edit Edita as credenciais criptografadas da aplicação. Documentação
rails secret Gera uma chave secreta para uso na aplicação. Documentação

Comandos de Migração

Comando Descrição Documentação
rails db:migrate:status Mostra o status das migrações. Documentação
rails db:rollback Reverte a última migração aplicada. Documentação

Como Saber Quais Migrações Foram Aplicadas

Para listar o status das migrações, use:

rails db:migrate:status

Para listar os arquivos de migração:

ls db/migrate/

Para verificar migrações aplicadas no banco de dados:

rails dbconsole

Em seguida, no console SQL, execute:

SELECT * FROM schema_migrations;

Mais Comandos

Para ver todos os comandos disponíveis no Rails, utilize:

rails --help
