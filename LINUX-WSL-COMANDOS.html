Principais Comandos Linux para WSL

Lista com os 100 principais comandos Linux para uso no Windows Subsystem for Linux (WSL), organizados por categoria.

Comandos Básicos

Comando Descrição Documentação
ls Lista arquivos e diretórios. Documentação
cd [diretório] Altera o diretório atual. Documentação
pwd Mostra o diretório atual. Documentação
mkdir [nome] Cria um novo diretório. Documentação

Manipulação de Arquivos

Comando Descrição Documentação
cp [origem] [destino] Copia arquivos e diretórios. Documentação
mv [origem] [destino] Move ou renomeia arquivos. Documentação
rm [arquivo] Remove arquivos. Documentação

Gerenciamento de Processos

Comando Descrição Documentação
ps Exibe processos em execução. Documentação
kill [PID] Finaliza um processo pelo ID. Documentação
top Monitoramento de processos em tempo real. Documentação

Rede

Comando Descrição Documentação
ping [host] Verifica a conectividade com um host. Documentação
curl [URL] Realiza requisições HTTP. Documentação
netstat Exibe conexões de rede ativas. Documentação

Mais Comandos

Para acessar a documentação de qualquer comando, você pode utilizar:

man [comando]

Exemplo:

man ls

Para ver todos os comandos disponíveis no sistema, use:

compgen -c
