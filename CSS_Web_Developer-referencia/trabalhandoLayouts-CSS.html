Position e Display

Controlam como os elementos são exibidos e posicionados na página.

Position

.relative { position: relative; top: 10px; left: 20px; } /* Relativo à posição original */ .absolute { position: absolute; top: 0; left: 0; } /* Relativo ao elemento pai posicionado */ .fixed { position: fixed; top: 0; left: 0; } /* Fixo na tela */

Display

.block { display: block; } /* Ocupa toda a largura */ .inline { display: inline; } /* Ocupa apenas o espaço necessário */ .inline-block { display: inline-block; } /* Combina características de block e inline */

Flexbox

Um modelo de layout para criar designs flexíveis e alinhados.

Exemplo Básico

.container { display: flex; justify-content: center; /* Alinha horizontalmente */ align-items: center; /* Alinha verticalmente */ }

Flex Direction

.container { flex-direction: row; /* Padrão: horizontal */ flex-direction: column; /* Vertical */ }

Grid Layout

Um sistema de layout bidimensional para criar designs complexos.

Exemplo Básico

.container { display: grid; grid-template-columns: repeat(3, 1fr); /* 3 colunas de tamanho igual */ gap: 10px; /* Espaço entre os itens */ }

Responsividade

Técnicas para adaptar o layout a diferentes dispositivos.

Media Queries

@media (max-width: 768px) { .container { flex-direction: column; /* Altera o layout para mobile */ } }
