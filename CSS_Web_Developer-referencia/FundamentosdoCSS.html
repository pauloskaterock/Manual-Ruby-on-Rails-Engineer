Fundamentos CSS

CSS (Cascading Style Sheets) é uma linguagem usada para estilizar elementos HTML.

Sintaxe Básica

h1 { color: blue; /* Propriedade: valor */ font-size: 24px; /* Tamanho da fonte */ }

Formas de Adicionar CSS

1. Inline: Direto no elemento HTML.

<h1 style="color: blue;">Título</h1>

2. Interno: Dentro da tag <style> no HTML.

<style> h1 { color: blue; } </style>

3. Externo: Em um arquivo `.css` separado.

<link rel="stylesheet" href="styles.css">

Estilização com CSS

Cores e Fundos

h1 { color: #ff5733; /* Hexadecimal */ background-color: rgba(255, 87, 51, 0.5); /* RGBA com transparência */ }

Fontes e Textos

p { font-family: 'Arial', sans-serif; font-size: 16px; font-weight: bold; }

Box Model

.box { width: 200px; padding: 20px; /* Espaço interno */ border: 2px solid black; /* Borda */ margin: 10px; /* Espaço externo */ }

Unidades de Medida em CSS

Unidades Absolutas

h1 { font-size: 24px; }

Unidades Relativas

.container { width: 80%; } /* Relativo ao elemento pai */ p { font-size: 1.5em; } /* Relativo ao tamanho da fonte do pai */ h2 { font-size: 2rem; } /* Relativo ao tamanho da fonte do <html> */ .hero { height: 100vh; } /* Relativo à altura da viewport */
