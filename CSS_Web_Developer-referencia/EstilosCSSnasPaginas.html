Pseudo-elementos e Pseudo-classes

Pseudo-elementos e pseudo-classes permitem estilizar partes específicas de elementos.

Pseudo-classes

a:hover { color: red; } /* Altera a cor ao passar o mouse */ input:focus { border-color: blue; } /* Altera a borda ao focar */

Pseudo-elementos

p::first-line { font-weight: bold; } /* Estiliza a primeira linha */ p::before { content: "→ "; } /* Adiciona conteúdo antes */

Referência: MDN Web Docs

Transições e Animações

Adicione interatividade e movimento aos elementos.

Transições

.btn { transition: background-color 0.3s ease; } .btn:hover { background-color: #0056b3; }

Animações

@keyframes slide { from { transform: translateX(0); } to { transform: translateX(100px); } } .box { animation: slide 2s infinite; }

Referência: MDN Web Docs

Boas Práticas e Convenções no CSS

Organize seu código para facilitar a manutenção.

BEM (Block Element Modifier)

.block { } /* Bloco */ .block__element { } /* Elemento */ .block--modifier { } /* Modificador */

Evite !important

Use especificidade ao invés de `!important`.
