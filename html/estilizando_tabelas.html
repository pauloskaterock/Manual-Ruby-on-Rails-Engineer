Você pode estilizar tabelas em HTML usando CSS para alterar a aparência visual das bordas, fundos, cores de texto e muito mais. Aqui estão algumas maneiras de estilizar tabelas: 1. Estilizando as Bordas:
table { border-collapse: collapse; border: 2px solid black; /* Define a cor e largura da borda da tabela */ } th, td { border: 1px solid black; /* Define a cor e largura da borda das células */ } 2. Alterando as Cores de Fundo e Texto:
th { background-color: #f2f2f2; /* Define a cor de fundo dos cabeçalhos das colunas */ } td { background-color: #ffffff; /* Define a cor de fundo das células de dados */ color: #333333; /* Define a cor do texto das células de dados */ } 3. Adicionando Preenchimento Interno às Células:
th, td { padding: 8px; /* Adiciona preenchimento interno às células */ } 4. Alterando o Estilo do Texto e da Fonte:
th, td { font-family: Arial, sans-serif; /* Define a família de fontes para o texto nas células */ font-size: 14px; /* Define o tamanho da fonte para o texto nas células */ font-weight: normal; /* Define o peso da fonte para o texto nas células */ } 5. Adicionando Espaçamento entre as Células:
table { border-spacing: 0; /* Define o espaçamento entre as células da tabela */ } 6. Estilizando Células de Cabeçalho Diferentemente:
th { font-weight: bold; /* Torna o texto nos cabeçalhos das colunas em negrito */ text-align: center; /* Alinha o texto dos cabeçalhos das colunas ao centro */ } Você pode personalizar ainda mais o estilo da tabela usando CSS de acordo com as necessidades de design específicas do seu projeto. Lembre-se de que esses são apenas exemplos básicos e você pode adicionar estilos adicionais conforme necessário para atender aos requisitos de design do seu site.
