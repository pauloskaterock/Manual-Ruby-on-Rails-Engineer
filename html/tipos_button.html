No HTML, a tag Este é o tipo padrão. Ele é usado para criar um botão comum que pode executar ações definidas por JavaScript quando clicado. Botão de Submit (Enviar): ----- Este tipo de botão é usado dentro de formulários. Quando clicado, ele envia o formulário para o servidor. Botão de Reset (Limpar): ----- Este tipo de botão é usado dentro de formulários. Quando clicado, ele limpa todos os campos do formulário para seus valores padrão. Botão de Imagem: ----- Este tipo de botão é semelhante a um botão de submit, mas em vez de um botão de texto, ele usa uma imagem como botão. Botão Oculto: ----- Este tipo de botão é usado para executar ações via JavaScript sem que o botão seja visível para o usuário. Botão de Reset (Resetar): Embora tecnicamente não seja uma tag
