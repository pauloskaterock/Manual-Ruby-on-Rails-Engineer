: Define a versão do HTML sendo usada, geralmente HTML5. : Define o início e o fim do documento HTML. : Contém informações sobre o documento, como título, meta tags, e referências a folhas de estilo. : Define o título da página, exibido na barra de título do navegador. <meta>: Fornece metadados sobre o documento, como conjunto de caracteres, descrição, palavras-chave, etc. <body>: Contém o conteúdo visível da página, como texto, imagens, links, etc. <h1> to <h6>: Define cabeçalhos de diferentes níveis, sendo <h1> o mais importante e <h6> o menos importante. <p>: Define um parágrafo de texto. <a>: Cria um link, permitindo a navegação entre páginas ou recursos. <img>: Incorpora uma imagem na página. <ul>: Define uma lista não ordenada. <ol>: Define uma lista ordenada. <li>: Define um item de lista, usado dentro de <ul> ou <ol>. <div>: Cria uma divisão ou seção na página, frequentemente usada para estruturar o layout. <span>: Define uma pequena seção de texto dentro de um elemento, muitas vezes usado com CSS para aplicar estilos. <form>: Cria um formulário para envio de dados. <input>: Define um campo de entrada em um formulário, como texto, senha, checkbox, etc. <button>: Cria um botão que pode ser usado para enviar um formulário ou acionar um script. <textarea>: Cria uma área de texto multi-linha. <select>: Cria um menu suspenso de opções. <iframe>: Permite incorporar conteúdo de outra página ou recurso. <hr>: Cria uma linha horizontal para dividir o conteúdo. <br>: Insere uma quebra de linha. <strong> e <em>: Define texto forte (negrito) e texto enfatizado (itálico), respectivamente. <cite>: Define o título de uma obra citada. <blockquote>: Define uma seção de texto que é uma citação de outra fonte.
