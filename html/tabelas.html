Neste exemplo: A tag é usada para criar a tabela. Cada linha da tabela é criada com a tag (linha da tabela). Os cabeçalhos da tabela são criados com a tag
(cabeçalho da tabela). As células da tabela são criadas com a tag (célula da tabela). O atributo border="1" define a largura da borda da tabela para torná-la visível. Este atributo é opcional e usado apenas para fins de visualização durante o desenvolvimento. Você pode adicionar quantas linhas e colunas forem necessárias à tabela para exibir seus dados tabulares de forma organizada. Além disso, você pode estilizar a tabela usando CSS para atender às suas necessidades de design específicas.
