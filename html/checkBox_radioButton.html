Neste exemplo: Checkbox: Cada checkbox pode ser selecionado ou desmarcado independentemente das outras opções. Os checkboxes são úteis quando o usuário pode selecionar várias opções.
Radio Button: Os radio buttons são mutuamente exclusivos, ou seja, apenas uma opção pode ser selecionada de cada vez. Eles são úteis quando o usuário deve selecionar apenas uma opção entre várias.
