A tag é uma das tags fundamentais em HTML usadas para criar tabelas. Ela define uma tabela que contém linhas e células. Aqui está a estrutura básica de uma tabela HTML:
Cabeçalho da Coluna 1 Cabeçalho da Coluna 2 Cabeçalho da Coluna 3
Dado da Linha 1, Coluna 1 Dado da Linha 1, Coluna 2 Dado da Linha 1, Coluna 3
Dado da Linha 2, Coluna 1 Dado da Linha 2, Coluna 2 Dado da Linha 2, Coluna 3
Neste exemplo:

<: Define a tabela. : Define uma linha na tabela.
: Define um cabeçalho de coluna. Geralmente usado na primeira linha da tabela para cabeçalhos. : Define uma célula de dados. Usado para cada célula de dados na tabela. É importante observar que a tag é usada para cabeçalhos de coluna, enquanto a tag é usada para células de dados regulares. A primeira linha da tabela geralmente contém cabeçalhos de coluna, enquanto as linhas subsequentes contêm dados. Você pode adicionar estilos CSS à tabela para personalizar sua aparência e layout, como largura da borda, espaçamento de células, cores de fundo, etc. Além disso, é possível adicionar atributos como border, cellpadding, cellspacing, entre outros, para controlar a aparência da tabela. /h3>
